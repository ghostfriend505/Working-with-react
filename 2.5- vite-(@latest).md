## What is Vite?

**Vite** (pronounced *veet*) is a modern build tool used to create fast React applications.  
It is developed by **Evan You** (creator of Vue.js) and is now the **recommended way** to start new React projects.

Vite focuses on:
-  speed
-  simplicity
-  modern development

## How to Create a React App Using Vite

    npm create vite@latest my-app
    cd my-app
    npm install
    npm run dev

### After running this, open browser and go to:

    https://localhost:5173

  Your app will run instantly
